# Context-aware-translation
This project is a context-aware academic translation system designed to preserve terminology consistency across long technical documents, especially in domains where literal translation fails. It builds a dynamic terminology memory from source texts and uses retrieval-based context selection to guide translation decisions and reduce ambiguity in domain-specific language.

The system is currently part of an ongoing academic research effort in collaboration with faculty members, exploring terminology-aware machine translation and retrieval-augmented context modeling for improving translation quality in technical and academic content.
